# 📚 اردو میں گائیڈ - بیگ ہیون پروجیکٹ

## پروجیکٹ کی معلومات

**نام:** BagHaven (بیگ ہیون)  
**مقصد:** پریمیم بیگز اور بیکپیکس بیچنا  
**تیاری میں استعمال شدہ ٹیکنالوجی:** Next.js, React, Tailwind CSS, Supabase

---

## 🛠️ استعمال شدہ ٹولز

### **فرنٹ اینڈ (جو صارف دیکھتا ہے)**
- **Next.js** - ویب سائٹ بنانے کا فریم ورک
- **React** - صارف انٹرفیس بنانے کے لیے
- **Tailwind CSS** - رنگ، ڈیزائن اور شکل دینے کے لیے
- **Lucide** - آئیکنز (چھوٹی تصویریں)

### **بیک اینڈ (سرور اور ڈیٹا بیس)**
- **Supabase** - ڈیٹا بیس اور یوزر لاگ ان
- **PostgreSQL** - اعلیٰ درجے کا ڈیٹا بیس

### **دوسری خدمات**
- **Supabase Storage** - صارف کی تصویریں محفوظ کرنا
- **GitHub** - کوڈ محفوظ کرنا

## 🔄 کیسے کام کرتا ہے؟

### **صارف کا سفر (User Journey)**

```
1️⃣ صارف ویب سائٹ کھولتا ہے
   ↓
2️⃣ گھر کا صفحہ دیکھتا ہے (app/page.js)
   ↓
3️⃣ پروڈکٹس کو صفحہ بندی کرتا ہے (app/products/page.js)
   ↓
4️⃣ ایک پروڈکٹ پر کلک کرتا ہے (app/products/[id]/page.js)
   ↓
5️⃣ "ٹوکری میں ڈالو" دبتا ہے
   ↓
6️⃣ ٹوکری دیکھتا ہے (app/cart/page.js)
   ↓
7️⃣ چیک آؤٹ کرتا ہے (app/checkout/page.js)
   ↓
8️⃣ ڈیٹا بیس میں آرڈر محفوظ ہوتا ہے
```

## ⚙️ کیسے بدلائں؟

### **گھر کے صفحے کا عنوان بدلیں**
فائل: `app/page.js`
ڈھونڈیں: `Carry Your World in Premium Style`
اس کی جگہ لکھیں: اپنا عنوان

### **ای میل بدلیں**
فائل: `app/contact/page.js`
ڈھونڈیں: `support@baghaven.pk`
اس کی جگہ لکھیں: آپ کی ای میل

### **نیا صفحہ شامل کریں**
مثال کے طور پر "بلاگ" صفحہ:
```
app/blog/page.js بنائیں
↓
http://localhost:3000/blog میں جائے گا
```

### **مینو میں نیا لنک شامل کریں**
فائل: `components/Navbar.js`
اس میں یہ لکھیں:
```javascript
{ name: 'بلاگ', path: '/blog' },
```

## 🛍️ ایک نیا پروڈکٹ کیسے شامل کریں؟

### **طریقہ 1: Supabase سے براہ راست**
1. اپنے Supabase اکاؤنٹ میں جائیں
2. `products` ٹیبل کھولیں
3. نیا قطار شامل کریں
4. نام، قیمت، تصویر بھریں
5. محفوظ کریں
6. ویب سائٹ پر خود بخود نظر آئے گا ✓

### **طریقہ 2: منتظم پینل سے**
1. منتظم کے طور پر لاگ ان کریں
2. `/admin/products` پر جائیں
3. فارم بھریں
4. جمع کریں

## 🎨 رنگ اور ڈیزائن

تمام رنگ اور ڈیزائن **Tailwind CSS** میں ہے۔

**اہم فائل:** `tailwind.config.js`

رنگ بدلنے کے لیے:
- `indigo-600` کو `blue-600` سے بدلیں
- یا `violet-600` استعمال کریں

**مثال:**
```javascript
className="bg-indigo-600"    // نیلا
className="bg-violet-600"    // بنفشی
className="bg-rose-600"      // گلابی
```

## 🐛 عام غلطیاں اور حل

### **غلطی: "Could not find avatar_url"**
**حل:** یہ کوڈ چلائیں Supabase میں:
```sql
ALTER TABLE public.users ADD COLUMN IF NOT EXISTS avatar_url TEXT;
```

### **غلطی: پروڈکٹ نہیں دکھائی دے رہے**
**حل:**
1. Supabase میں منسلک ہیں؟
2. `products` ٹیبل میں ڈیٹا ہے؟
3. `.env.local` درست ہے؟

### **غلطی: لاگ ان نہیں ہو رہا**
**حل:**
1. ای میل درست ہے؟
2. پاس ورڈ درست ہے؟
3. Supabase میں یہ صارف ہے؟
